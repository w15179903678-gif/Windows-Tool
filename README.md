# Windows-Tool
一、窗口自动化助手 PRO (Window Automation Assistant PRO)”，是一个基于 WPF (C#) 开发的 Windows 桌面连接与自动化工具。
它的核心功能是录制并回放针对特定窗口（如雷电模拟器或其他游戏窗口）的鼠标操作，且支持在后台执行，不占用真实的鼠标指针。

1. 项目核心定位
该工具主要用于解决需要重复执行点击或拖拽任务的情景。它不仅能记录你的操作，还能通过 Win32 API 将这些操作精确地“投递”给目标窗口。

2. 主要功能模块
智能窗口绑定：
   通过窗口标题（如“雷电模拟器”）精确锁定目标句柄（HWND）。能自动识别模拟器的渲染层（如 TheRender 层），确保点击坐标的精确性。\r\n
高精度录制引擎：
   利用 全局鼠标钩子 (Low-Level Mouse Hook) 实时捕获点击和拖拽动作。自动计算动作之间的时间间隔 (Delay) 和拖拽的持续时间 (Duration)。\r\n
后台静默回放：
   采用 PostMessage 技术。这意味着在自动化任务运行时，你可以继续使用电脑做其他事情，脚本会在后台模拟消息发送给目标窗口。支持平滑拖拽模拟，通过分段插值算法模拟真实的鼠标滑动轨迹。\r\n
任务管理与持久化：
   支持将录制的动作序列保存为 JSON 配置文件，方便下次直接加载使用。提供循环模式，可设置轮次间隔，实现全天候自动化。

总结来说，这是一个麻雀虽小五脏俱全的自动化助手，非常适合作为游戏辅助工具、自动化测试工具或重复性办公任务的脚本引擎。
